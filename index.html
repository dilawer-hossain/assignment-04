Types of CSS

1) Inline CSS
2) Internal CSS
3) External CSS (External CSS is good to use)

Inline CSS

=) Inline css is used in the same line of tag/element
=) In inline css style has been used as style attribute
=) Name of the attribute will be style
=) e.g. style="color:red"

Internal CSS

=>Internal css has been used in the same page of html coding.
=> In the head tag there will be another tag of style
=>< style>
{
color:red;
}
< /style>

External CSS

1) Need to make file and the file extension name will be style.css
2) a link have to be made in the head tag naming < link rel="stylesheet" href="file path of style.css file">
